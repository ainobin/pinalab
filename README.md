# PinaLab - Sustainable Fashion from Pineapple Leaves

## About PinaLab
PinaLab transforms agricultural waste into premium sustainable fashion. We collect pineapple leaves from local farmers, process them in our in-house factory, and create eco-friendly vegan leather and textiles.
